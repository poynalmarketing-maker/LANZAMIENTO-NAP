SUBIR A GITHUB PAGES

1. Crear un repositorio nuevo.
2. Subir TODOS los archivos de esta carpeta.
3. Ir a Settings > Pages.
4. En "Branch" seleccionar main y "/root".
5. Guardar.
6. Tu página quedará publicada automáticamente.

Para agregar productos nuevos:
- Abrí el archivo productos.js
- Copiá una línea y cambiá nombre, imagen y link.
